Key Benefits

1. Depth & Perspective Information

2. Faster Design Approvals

Communicate design effectively in both external presentations and internal coordinations

3. Intuitive Manipulation of BIM

VDC Oriented Features

Measurements

Accurate BIM measurements made with just a intuitive point and shoot motion.

Multi-User

Coordinate BIM across the internet in Virtual Reality, up to 8 VR users at once.

Daylight Analysis

Set time of day and sun position in order to experience your building in dusk to dawn conditions.

BIM Object Data

Retrieve any element object data easily in both VR and desktop interfaces.

Annotations & Comments

Markup an area with hand motions and use speech-to-text to tag an object for reiteration.

Render Modes

Switch render modes depending on the discussion for further design communication.

Merge BIM Models

Combine models from multiple BIM software into a single VR interactive model.

Desktop Friendly

All features are that are available in VR are also enabled for mouse and keyboard usage.
